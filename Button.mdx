## Button Components

Learn how to use and customize button components in your project.

### Basic Button

<AccordionGroup>
    <Accordion icon="button" title="Basic Button">
        To create a basic button, use the following code:
        ```jsx
        <button>Click me</button>
        ```
    </Accordion>
    <Accordion icon="customize" title="Custom Button Styles">
        You can customize the button styles using CSS:
        ```css
        button {
            background-color: blue;
            color: white;
            padding: 10px 20px;
            border: none;
            border-radius: 5px;
        }
        ```
    </Accordion>
</AccordionGroup>

### Button with Icon

<AccordionGroup>
    <Accordion icon="icon" title="Button with Icon">
        To add an icon to your button, use a library like FontAwesome:
        ```jsx
        <button>
            <i className="fa fa-check"></i> Submit
        </button>
        ```
    </Accordion>
    <Accordion icon="customize" title="Custom Icon Button Styles">
        Customize the icon button styles:
        ```css
        button {
            display: flex;
            align-items: center;
        }
        button i {
            margin-right: 5px;
        }
        ```
    </Accordion>
</AccordionGroup>

### Button Group

<AccordionGroup>
    <Accordion icon="group" title="Button Group">
        To create a group of buttons, use the following code:
        ```jsx
        <div className="button-group">
            <button>Button 1</button>
            <button>Button 2</button>
            <button>Button 3</button>
        </div>
        ```
    </Accordion>
    <Accordion icon="customize" title="Custom Button Group Styles">
        Customize the button group styles:
        ```css
        .button-group button {
            margin-right: 10px;
        }
        ```
    </Accordion>
</AccordionGroup>
